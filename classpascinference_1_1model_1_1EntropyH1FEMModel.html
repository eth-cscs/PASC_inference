Detailed Description

template<class VectorBase>
class pascinference::model::EntropyH1FEMModel< VectorBase >

time-series model with quadratic penalty time-space regularisation and Entropy-based Theta model.

Member Enumeration Documentation

template<class VectorBase >
enum pascinference::model::EntropyH1FEMModel::GammaSolverType

type of solver used to solve inner gamma problem

The inner problem leads to QP with equality constraints and bound constraints

Enumerator
GSOLVER_AUTO 

choose automatic solver

GSOLVER_SPGQP 

CPU/GPU implementation of Spectral Projected Gradient method

GSOLVER_SPGQP_COEFF 

CPU/GPU implementation of Spectral Projected Gradient method with special coefficient threatment

GSOLVER_PERMON 

PERMONQP solver (augumented lagrangians combined with active-set method)

GSOLVER_TAO 

TAO solver

template<class VectorBase >
enum pascinference::model::EntropyH1FEMModel::ThetaSolverType

type of solver used to solve inner theta problem

Maximum entropy - nonlinear optimization problem / system of nonlinear equations with integrals

Enumerator
TSOLVER_AUTO 

choose automatic solver

TSOLVER_ENTROPY_DLIB 

CPU/GPU implementation of Spectral Projected Gradient method

TSOLVER_ENTROPY_SPG 

CPU/GPU implementation of Spectral Projected Gradient method with special coefficient threatment
