Detailed Description

template<class VectorBase>
class pascinference::algebra::SimplexFeasibleSet_LinEqBound< VectorBase >

simplex feasible set defined by linear equality constraints and bound constraints

Provides structure for manipulation with feasible set in form

= x R^{KT}: Bx = c x 0
