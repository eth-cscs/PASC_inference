GitHub and doxygen

INITIALIZATION of gh-pages and first HTML page on github
-------------------------------------------------------------------------
- clone repository
git clone https://github.com/lukaspospisil/pizza.git
- create gh-pages branch (there will be stored html pages for git)
git checkout --orphan gh-pages
- delete all content in git repository (in active branch, i.e. in gh-pages)
git rm -rf .
- create sample page
echo "Hello world, this is our html fun!" > index.html
- add new file to git
git add index.html
- commit changes
git commit -a -m "First pages commit"
- push changes (but you deleted .git stuff from the folder, therefore it has to be set up again)
git push --set-upstream origin gh-pages
git push origin gh-pages
- open a web browser and test it:
http://lukaspospisil.github.io/pizza/
- from official Git documentation:
'Note that Pages are always publicly accessible when published, even if their repository is private.'


Branch gh-pages in stand-alone directory html
-------------------------------------------------------------------------
- switch to master branch
git checkout master
- create folder for documentation
mkdir html
- add html directory to .gitignore
- commit and push changes
- clone the whole project (master+gh-pages) into a new subdirectory
cd html/
git clone https://github.com/lukaspospisil/pizza.git .
- switch to branch gh-pages
git checkout origin/gh-pages -b gh-pages
- afterwards, there should be only index.html page created in the first step
- get rid of master branch (in this folder - there will be only gh-pages, to be sure that we will not push into master from this folder)
git branch -d master
- control active branches using
git branch
- get back to main project directory and check branches here
cd ..
git branch


DOXYGEN for generating HTML
-------------------------------------------------------------------------
- create a folder for documentation
mkdir docs
cd docs
- initiate doxygen
doxygen -g config.dox
- config.dox was created, edit it and check 
GENERATE_HTML = YES
GENERATE_LATEX = NO
- add "doxygen_sqlite3.db" to .gitignore (I don't know what to do this generated file)


Generate Doxygen files after changes in code
-------------------------------------------------------------------------

git push origin master
doxygen docs/config.dox
cd html
git push origin gh-pages
