Detailed Description

template<class VectorBase>
class pascinference::algebra::GeneralFeasibleSet< VectorBase >

general class for manipulation with feasible sets

Parent class for manipulation with feasible sets. All specific feasible set implementations should be defined as inherited classes from this class.
